
1.安装python3，当前安装版本为python 3.13.3



2.创建虚拟环境
MAC:
创建虚拟环境：python3 -m venv venv
激活虚拟环境：source venv/bin/activate

windows：
创建虚拟环境：python3 -m venv venv
激活虚拟环境：.\venv\Scripts\activate


3.初始化安装依赖：pip install -r requirements.txt



4.启动：
激活虚拟环境之后：python3 manage.py runserver
然后进/template/vue里  npm run dev 启动
打开网页


提交代码：
1.同步远程仓库：git pull
2.查看当前修改文件：git status
3.提交代码到本地git缓存区：git add . 或者git add 文件名1 文件名2 ...
4.推送代码到本地git库：git commit -m "提交代码备注"  （git commit 文件名 -m "提交代码备注"）
5.提交本地代码到原创仓库：git push   （git push 主机名字 远程分支名字）
